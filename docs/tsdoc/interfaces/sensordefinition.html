Interface SensorDefinition

Static definition of a sensor for use in SensorController. This is used in the options of the controller to register the sensors directly at creation. The 'sensors' option of the SensorController should contain an array of SensorDefinitions.

Each definition should contain the Sensor object, the hardware pin that it is connected to, and the type of the IO communication that it has.

Defines whether and how the observations should be automatically published from the controller. This is used together with the samplingInterval value. If it is not set to none, samplingInterval must be set to positive value.

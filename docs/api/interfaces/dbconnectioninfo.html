Interface DbConnectionInfo

Describes information used to connect to a specific database server using a specific database adapter.

DbConnectionInfo objects conform to JSON format in order to be serializable (see interface DatabaseOptions).

adapter

adapter: string
  • Defined in src/runtime/configuration.ts:252

The name of the adapter used to interact with a specific database server.

The name of the adapter specified here must be associated with a built-in or custom adapter type by using the DbAdapterFactory.registerAdapter method or by specifying the adapter type on first use when creating a new DbContext or DbLocalContext.
